Índice
1. Propósito e como usar
Este documento (AA) é o catálogo de produção de arte de Gênese: a lista completa de tudo que precisa ser desenhado/animado, organizada por categoria, com prioridade, e com um checklist marcável ao fim de cada bloco. Ele foi pensado para ser entregue a quem for produzir o design final (incluindo o Claude Design), junto das suas referências.
Fluxo de uso: (1) você reúne as referências por categoria no documento de Design Visual (DV), nos espaços tracejados; (2) usa este AA como a lista mestra do que pedir; (3) marca cada item à medida que for concluído. O DV define o ESTILO; este AA define a QUANTIDADE e a LISTA.
Referências cruzadas: a coluna &apos;Origem&apos; de cada tabela aponta para o documento que justifica aquele asset (GDD = visão; M- = mecânica; E- = etapa de construção; DV = direção de arte). Assim ninguém produz arte sem saber para que sistema ela serve.

11. Como referenciar com o resto do projeto
Este AA conversa com os demais documentos assim:
DV (Design Visual): define o ESTILO de tudo que está listado aqui. Cole suas referências no DV; produza pela lista do AA.
GDD: a visão (§8 visual, §9 UI) que justifica as categorias.
Documentos M- (mecânicas): dizem para que sistema cada asset serve (coluna &apos;Origem&apos;).
Documentos E- (etapas): E10 implementa o render desses assets; E11/E12 cobrem os assets de loja.
Sugestão prática para o pedido ao Claude Design: entregue, por categoria, (a) a referência reunida no DV, (b) a linha correspondente deste AA, e (c) a nota &apos;modular&apos; quando aplicável. Isso dá estilo + lista + restrição técnica num pacote só, sem ambiguidade.
